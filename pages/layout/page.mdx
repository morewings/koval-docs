# Page layout

Page layout provides utility components to organize arbitrary web page sections: `Header`, `Main`, `Sidebar` and `Footer`.

<Callout type="info" emoji="💡️">
    More examples of `Layout` are available at [Koval UI
    Storybook](https://morewings.github.io/koval-ui/?path=/docs/layout-page--docs).
</Callout>

## Code example

```tsx
import {Main, Footer, Sidebar, Header, Content, Page} from 'koval-ui';
return (
    <Page>
        <Header>This is Header</Header>
        <Content>
            <Sidebar xs={3}>This is Sidebar</Sidebar>
            <Main>This is Main</Main>
        </Content>
        <Footer>This is Footer</Footer>
    </Page>
);
```

## Configuration

### Page

`Page` component serves as a container and works the same as [Grid](/layout/grid#grid) component.

<Callout type="info" emoji="💡️">
    See [all Page
    props](https://morewings.github.io/koval-ui/?path=/docs/layout-page--docs#page-props).
</Callout>

### Content

`Content` wraps page content and sidebar. Works the same as [Row](/layout/grid#row) component.

<Callout type="info" emoji="💡️">
    See [all Content
    props](https://morewings.github.io/koval-ui/?path=/docs/layout-page--docs#content-props).
</Callout>

### Page content

`Sidebar` and `Main` work the same as [Col](/layout/grid#col) component.

<Callout type="info" emoji="💡️">
    See all
    [Sidebar](https://morewings.github.io/koval-ui/?path=/docs/layout-page--docs#sidebar-props) and
    [Main](https://morewings.github.io/koval-ui/?path=/docs/layout-page--docs#main-props) props.
</Callout>

### Footer

`Footer` component always sticks to the bottom of the page.

See [all Footer props](https://morewings.github.io/koval-ui/?path=/docs/layout-page--docs#footer-props).
